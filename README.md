# **IoT-Integrated AI System for Microplastic Detection in Water 🌊🤖**

A Raspberry Pi 4–powered edge-AI system that captures water sample images and detects microplastics using a lightweight deep-learning model. Everything runs locally on the Pi, offering a portable, low-cost approach to real-time microplastic detection without cloud support.

---

## 📸 **Project Overview**

Microplastics are nearly invisible and usually require expensive lab equipment to detect.  
This project shows how **IoT hardware + edge AI** can work together to spot them using only:

- A **Raspberry Pi 4**
- A **Pi Camera Module**
- A **deep-learning model** (CNN/YOLO)
- Python-based image processing

The Pi captures the water sample image, runs the model locally, and displays detection results instantly.

---

## 🚀 **Features**

- **Edge-Based AI Processing** – All inference runs directly on Raspberry Pi 4  
- **Image Capture Pipeline** – Uses Pi Camera Module for real-time imaging  
- **Deep Learning Detection** – Identifies microplastic shapes/patterns  
- **Offline & Cloud-Free** – Works anywhere without internet  
- **Low Cost & Portable** – Suitable for field research and academic use  

---

## 🛠 **Tech Stack**

- **Hardware:** Raspberry Pi 4, USB Camera 
- **Languages:** Python  
- **Libraries:** OpenCV, NumPy, TensorFlow/PyTorch, CUDA
- **Tools:** MakeSenseAI 
- **Model:** YOLOv5 Nano
- **OS:** Raspberry Pi OS
